gifs app

Es una aplicación en la cual se trabaja mediante comunicación entre componentes, y se consume un api externo. En la cual la idea es mostrar una de interfaz de búsqueda que también muestre un historial de lo que se ha buscado.

Filtro pipe

app de filtrado

Esta es una aplicación simple, que filtra diferentes valores por medio de un pipe y ngmodel, se consume un API externa simple que trae productos en inglés, la idea es verificar que el filtrado funciona.

FormApp

app de formularios reactivos

Esta es una aplicación con formularios reactivos que incluyen interfaces simples, dinámicas y switches. Este enfoque mejora la interacción del usuario y la eficiencia en la gestión de datos, demostrando las capacidades avanzadas de Angular para crear soluciones sofisticadas y altamente interactivas.
